Voice for Visualisation

This interface allows you to use voice commands to manipulate the visualisation to the right.

Want to see the sales for a particular country?
Use commands like:
"Show Portugal", "How were the sales in Spain?", "What happened in Britain?"

Want to reset the visualisation?
Use commands like:
"Let's start over", "Reset" or "Clear"
